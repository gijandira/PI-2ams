# 🧩 Autim - Projeto Integrador
> **Projeto Integrador | 2º ADS-AMS - FATEC TAUBATÉ**

## 🎯 Objetivo do Projeto
*Sistema voltado para auxiliar no desenvolvimento e acompanhamento de pessoas no espectro autista.*

---

## 📝 O Desafio
*Desenvolver uma ferramenta tecnológica que auxilie [descreva aqui o problema principal, ex: a comunicação entre pais e terapeutas / o aprendizado lúdico], visando melhorar a qualidade de vida e a autonomia dos usuários através de uma interface intuitiva e funcional.*

---

## 📌 Escopo do Projeto

> **Requisitos Funcionais**
* **Cadastro de Usuários:** Perfis diferenciados para responsáveis, profissionais e alunos.
* **[Funcionalidade 2]:** Descrição rápida (ex: Mural de atividades diárias).
* **[Funcionalidade 3]:** Descrição rápida (ex: Relatórios de progresso).
* **Alertas e Notificações:** Avisos sobre horários de medicamentos ou atividades.

> **Requisitos Não Funcionais**
* **Acessibilidade:** Interface adaptada com cores e fontes que evitam sobrecarga sensorial.
* **Segurança:** Proteção de dados sensíveis conforme a LGPD.
* **Performance:** Carregamento rápido de mídias e atividades.

---

## 📅 Planejamento de Entregas (Sprints)
- **SPRINT 1:** Planejamento e Prototipagem ✅
- **SPRINT 2:** Desenvolvimento do Banco de Dados e Login 🔄
- **SPRINT 3:** Implementação das Funcionalidades Principais ⏳
- **SPRINT 4:** Testes e Refinamento ⏳
